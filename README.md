# Credit-Risk-Model-using-Machine-Learning
Finance, a Non-Banking Financial Company (NBFC), to develop a sophisticated credit risk model. This model will include a credit scorecard that categorizes loan applications into Poor, Average, Good, and Excellent categories, based on criteria similar to the CIBIL scoring system.
